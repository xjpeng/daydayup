  Y轴向屏幕下方变大.

### 角度与弧度

数学函数计算所使用的角度单位为 弧度

Math类方法和Canvas类方法 使用弧度

CSS transform 可以使用弧度和角度带上单位即可

弧度 = 角度*Math.PI/180

角度 = 弧度*180/Math.PI

1弧度 ≈= 57度

1度   ~= 0.017弧度


### 两点间距离

 //点A(x1,y1)
 //点B(x2,y2)
 
 var dx = x2 - x1;
 var dy = y2 - y1;
 var distance = Math.sqrt(dx*dx+dy*dy);

### 单位圆

半径为1的圆,斜边为1

sin(A) = ∠A的对边/斜边    sin(A) = ∠A的对边

cosin(A) = ∠A的邻边/斜边  cosin(A) = ∠A的邻边

sin(A) = Y方向长度

cosin(A) = X方向长度

### 矢量

矢量是有方向有大小的量

矢量运动,必须要知道角度和速度
```javascript

 //假定角度为30度
 //速度为10
 var angle = 30;
 var speed = 10;
 var radian = angle*Math.PI/180;
 var vx = Math.cos(radian)* speed;
 var vy = Math.sin(radian)* speed;

 element.x = vx;
 element.y = vy;
  
```
